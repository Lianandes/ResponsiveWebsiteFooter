# Responsive Website Footer

Projeto desenvolvido para praticar a criação de rodapés responsivos para websites utilizando HTML e CSS.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em desenvolvimento front-end, responsividade e construção de componentes reutilizáveis para interfaces web.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Footer responsivo
- Layout moderno
- Organização de conteúdo
- Interface adaptável
- Estrutura visual organizada

## 📚 Conceitos praticados

- Responsividade
- Flexbox
- Media Queries
- Estruturação de layouts
- Estilização de componentes
